# Fruit-And-Vegetable-Detection

A dataset of images containing 120 fruits and vegetables.

Detecting the fruits and vegetables using CNN.

Download the dataset from here:
https://www.kaggle.com/moltean/fruits


This project used CNN concepts to train the model.
Flask and HTML are used to build the frontend part.

The output page will be the Webpage which will classify the image uploaded by the User.
